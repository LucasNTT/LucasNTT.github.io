Matrix Calculator

Matrix Calculator is a C# program designed for working with matrices using LaTeX syntax.
Enter your commands in simple math-mode LaTeX on the left, and the program displays the computed result on the right.
It supports variables, formulas, and advanced matrix operations.
The tool has been used by the author to validate FFT algorithms through matrix factorizations, as described among others here, here, here and here.
